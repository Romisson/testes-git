Mini Curso de Markdown para README

1. Títulos:
   - Use o símbolo `#` seguido de um espaço para criar títulos e subtítulos:
     # Título Principal
     ## Subtítulo
     ### Sub-subtítulo

2. Ênfase no texto:
   - Negrito: **texto**
   - Itálico: *texto*
   - Negrito e Itálico: ***texto***

3. Listas:
   - Ordenada:
     1. Item 1
     2. Item 2
   - Não ordenada:
     - Item A
     - Item B

4. Links:
   - Formato: [Texto do Link](URL)
   - Exemplo: [Visite meu repositório](https://github.com/romisson-oliveira)

5. Imagens:
   - Formato: ![Texto Alternativo](URL-da-Imagem)
   - Exemplo: ![Logo GitHub](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)

6. Blocos de Código:
   - Inline: `código`
   - Bloco:
     ```
     console.log('Olá, mundo!')
     ```

7. Tabelas:
   - Formato:
     | Coluna 1 | Coluna 2 |
     |----------|----------|
     | Valor 1  | Valor 2  |

8. Linhas Horizontais:
   - Use `---`:
     ---
